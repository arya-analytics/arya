The Synnax console is a desktop application for visualizating data, controlling
hardware, and managing a Synnax cluster. This page will get you started with installing 
the console, and point you to the right resources for learning how to use it.

## Install

Install the console by downloading the latest release for your operating system.

## Next Steps

Now that you've installed the console, [connect it to a Synnax cluster](/console/connect-a-cluster)
to start visualizing data and controlling hardware.
