Workspaces in Synnax are collections of visualizations that can be used to load
certain views. For instance, you could be monitoring an entire plane on the
Synnax cluster, but have one workspace specifically for the fuel system and
another workspace for aerodynamic sensors.

<Divider.Divider direction="x" />

## Creating a New Workspace
<Text.Text level="p">
To create a new workspace, you can go into the top-left and click on the
__Workspace__ <Icon.Workspace/> icon. This allows you to add a new workspace.
</Text.Text>
